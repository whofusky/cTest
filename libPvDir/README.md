# libPvDir说明

此目录下的源码是为光伏2.0项目获取公共文件夹编写的接口

## 文件夹下目录结构说明

```
./
├── bin
│   ├── CDUS         #make myexe=CDUS编译出的可执行文件
│   └── SCADA-CFG    #make myexe=SCADA-CFG编译出的可执行文件
├── include
│   ├── noRecursive  #额外引用头文件目录，暂时无用
│   └── PvDir.h      #源码头文件
├── lib              #编译需要引用的库文件目录，暂时无用
├── main.cpp         #为了测试写日志功能代码写的临时测试代码
├── Makefile         #makefile文件，方便将源码封装成.a或.so的库文件
├── out_lib
│   └── libPvDir.a   #用make mylib=libPvDir.a编译出的静态库
├── README.md        #markdown 格式的readme文件
├── src
│   └── PvDir.cpp     #源码
└── tmpFile           #临时目录，暂无用

```

## 关于libPvDir.a的说明

```

/******************************************************************************/
// 说明:
//      此接口实现的功能为实现光伏2.0项目中取公共配置文件夹名而写
//      1. 调用此接口的程序运行在如下两种目录结构中
//        (1) /一级目录名/项目名/id1/程序文件夹名
//        (2) /一级目录名/项目名/id1/id2/程序文件夹名
//      2. 调用GetPvPubDir接口获取"/一级目录名/项目名/id1/pubfile"目录下
//         公共文件夹的路径
//      3. 调用GetPvPubFile接口获取"/一级目录名/项目名/id1/pubfile/cfg"目录下
//         公共配置文件名(带路径)
//      4. 调用GetPvRunPar接口获取:运行程序相关路径参数

/******************************************************************************/

```

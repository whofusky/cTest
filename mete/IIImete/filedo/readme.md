filedo：此目录是模拟气象下载器各文件目录之用的

```
---------------------------------------------
    do:             正在处理的文件之文件夹
    -----------------------------------------
    tmp:            临时文件夹
    -----------------------------------------
    winplan         测风计划目录
    winplan/do      正在生成计划的目录
    winplan/yes     计划生成完毕目录
    -----------------------------------------
    down:           ftp下载后的文件夹
    down/ser1       正在下载的目录
    down/ser2       正在下载的目录
    down/yes        下载完成文件移此目录
    -----------------------------------------
    statList:       业务清单提供情况生成目录
    statList/trig   业务清单提供情况 触发目录
    
    -----------------------------------------
    middle:         III区摆渡中间区
    up:             ftp上传文件区
    -----------------------------------------
    health          健康报告相关目录
    health/do       健康报告生成目录
    health/yes      健康报告完成目录 也是综合报告触发目录
    health/frmII    场内上传的健康报告 也是综合报告触发目录
    health/integ    综合健康报告生成目录
    health/up       健康报告上传目录

    -----------------------------------------
    
```
